# MTG Network Analysis

Static GitHub Pages site for interactive MTG draft and cube archetype analysis.

## Local structure
- `index.html`: top-level landing page
- `tarkir/`: Tarkir Dragonstorm interactive analysis
- `lorwyn/`: Lorwyn Eclipsed interactive analysis
- `cube/`: Powered Cube interactive analysis
- `reports/`: supporting PDFs

## GitHub Pages
After pushing this repo:
1. Open the repo on GitHub.
2. Go to `Settings` > `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save. GitHub will publish the site at `https://<your-username>.github.io/mtg-network-analysis/`.

GitHub Pages can take a minute or two to finish the first deployment.
