Privacy Policy

How this site measures usage

Last updated: March 8, 2026.

This site is a static analytics site about Magic: The Gathering limited formats. It is hosted on GitHub Pages. When analytics are enabled in the site configuration, the site uses third-party analytics tools to understand how visitors use the site and where the experience should improve.

What we collect when analytics are enabled

  • Basic site usage information such as page views, visits, referrers, browser/device information, and approximate location inferred by analytics providers.
  • Interaction data such as which format pages people open, which reports are clicked, and which on-site controls are used.
  • Behavioral analytics such as clicks, scroll depth, engagement time, and session playback/heatmap data.

Analytics providers

  • Google Analytics 4 is used for traffic, sessions, engagement, and custom event measurement.
  • Microsoft Clarity is used for heatmaps, click analysis, and session recordings.

Why we collect it

  • Understand which sets, archetypes, and features people care about.
  • Find friction points in navigation and interaction.
  • Improve the site and decide what to build next.

How to limit tracking

You can block analytics by using browser privacy settings, tracker blocking extensions, or content blockers. Google also offers an official Analytics opt-out add-on: tools.google.com/dlpage/gaoptout.

Data sharing

This site uses external analytics vendors listed above. Data may be processed by those vendors according to their own terms and privacy policies.
